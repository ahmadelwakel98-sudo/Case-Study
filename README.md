# Case-Study
Sales Analysis (Excel)
Overview
This project focuses on analyzing raw sales data using Microsoft Excel to generate meaningful insights. The main goal was to transform unstructured data into a clean, analyzable format and build dynamic reports to monitor sales performance and support decision-making.
Data Preparation & Cleaning
•	Reviewed raw dataset and identified inconsistencies and missing values
•	Handled data cleaning by removing duplicates and correcting errors
•	Standardized data formats (dates, numbers, categories)
•	Ensured data accuracy and consistency before analysis
Data Transformation
•	Used Excel functions such as INDEX, MATCH, IF, and logical formulas
•	Created calculated columns to support analysis
•	Structured data into analysis-ready format
•	Built helper columns for KPI calculations
Analysis & KPIs
•	Calculated Total Sales and Revenue metrics
•	Measured Growth Percentage across time periods
•	Analyzed Month-to-Date (MTD) performance
•	Compared actual performance vs targets
•	Tracked time progress vs sales progress
Reporting & Visualization
•	Built dynamic reports to summarize sales performance
•	Created summary tables for decision-making
•	Highlighted top-performing customers
•	Identified low-performing segments and gaps
•	Presented trends in a clear and simple format
Outcome
The project resulted in a structured and automated reporting system that provides clear visibility into sales performance, helping stakeholders identify trends, opportunities, and performance gaps.
